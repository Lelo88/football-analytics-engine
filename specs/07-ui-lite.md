# UI Lite Spec

## Objective
Provide a lightweight custom UI only after the data model, ingestion, query layer and API are stable.

## Scope
- server-rendered pages
- HTMX filters
- Bootstrap styling
- Chart.js graphs

## Functional Requirements
- team summary page
- filters for latest N and home/away
- summary charts and tables

## Non-Functional Requirements
- simplicity
- low maintenance
- no SPA complexity
- architecture compliance

## Acceptance Criteria
- 3 to 5 useful views exist
- UI reflects API/query data correctly
- no business logic leaks into templates
