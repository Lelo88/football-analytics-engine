# Metabase Dashboards Spec

## Objective
Provide immediate visual value using Metabase over PostgreSQL.

## Scope
- Metabase connection
- saved questions
- initial dashboards
- dashboard filters

## Functional Requirements
- team overview dashboard
- league overview dashboard
- filters for team and season where possible
- saved questions for team form, over/under and goals summary

## Non-Functional Requirements
- clarity
- reproducibility
- low maintenance
- usefulness without custom frontend

## Acceptance Criteria
- dashboards open successfully
- filters return meaningful values
- dashboard results align with query layer output
